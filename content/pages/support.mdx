## Support

If you run into problems while using Gamiary, start with the checks below before contacting support.

## Common Issues

### A game does not load

- Refresh the page and try again.
- Check whether your browser blocks third-party content.
- Try the same page in another browser or device.

### The page layout looks wrong

- Clear browser cache and reload the page.
- Make sure JavaScript is enabled.
- Update your browser to a recent version.

### A game link or category is incorrect

- Send us the page URL and describe the issue.
- If possible, include screenshots or device/browser details.

## Contact Support

If the issue remains, contact us at:

**Email**: [support@your-domain.com](mailto:support@your-domain.com)
